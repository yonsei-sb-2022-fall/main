# 빅데이터 분석 2

## Course Description

이번 과정에서는 다양한 머신러닝 알고리즘을 사용하여, 다양한 데이터를 활용한 모델을 만드는 과정에 대해서 배웁니다.

## Prerequisites:

파이썬 기본

## Learning Outcomes

학기말이 되면 여러분들은 다음과 같은 것들을 할 수 있게됩니다.

1. 모델 훈련 및 검증
1. 데이터의 형태별 다양한 모델 훈련 적용
1. 모델 배포
